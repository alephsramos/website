Laboratório virtual de treinamento de pesquisa biomédica

Redes sociais do projeto

Este projeto tem como principal objetivo proporcionar suporte através de um jogo de realidade virtual, uma vez que os ambientes de treinamento específicos para a área de biomedicina apresentam elevados riscos, baixa fidelidade, são de difícil acesso e onerosos. Para profissionais atuantes nesse campo, o nosso projeto apresenta propostas inovadoras, tais como um laboratório virtual de treinamento que abrange técnicas e procedimentos relacionados ao manuseio de instrumentos, seguindo rigorosamente as normas estabelecidas pela biossegurança.
